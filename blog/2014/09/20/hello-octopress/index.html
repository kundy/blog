使用octopress为github输出blog页面

1、先clone到本地

https://github.com/imathis/octopress.git

2、当然也要安装好ruby和devkit

3、因为ruby的软件源在国内被屏蔽了，可以换成taobao的

  • gem sources -a http://ruby.taobao.org/
  • gem sources -r http://rubygems.org/
  • gem sources -l

4、修改配置

进入octopress目录，修改里面的gemfile文件，将socrce修改成 source “https://ruby.taobao.org”

5、安装bundler

bundler管理项目中gem的依赖项，通过以下命令会自动下载和安装依赖包 gem install bundler bundle install

6、安装octopress默认主题

rake install

5、生成和预览博客。

rake generate rake preview 然后在浏览器中输入http://localhost:4000
